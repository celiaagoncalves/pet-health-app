O que faz

Funcionalidades que correm 100% no teu telemóvel, sobre uma base de dados local (Hive).

🐾

8 espécies + personalizada

Cão, gato, ave, coelho, hamster, peixe, réptil — ou qualquer outro animal via opção "Outro" com nome livre.

💉

Registos clínicos

Vacinas, desparasitação, consultas, cirurgias e exames com notas e data da próxima ocorrência.

📒

Caderneta digital

Microchip, seguro, contacto do veterinário, peso, esterilização, alergias e condições médicas.

🔔

Lembretes automáticos

Notificações locais 3 dias antes e no próprio dia de cada tratamento agendado, às 09:00.

📊

Estado num relance

Vê o que está em atraso, o que está para a próxima semana e o que vem mais à frente, agrupado por animal.

🌍

Bilingue PT/EN

Português (por defeito) e inglês, alternáveis em runtime a partir do separador Definições. Sem reiniciar a app.

🎂

Idade automática

Calculada à granularidade do dia: "5 dias", "1 mês", "2 anos". A data de nascimento por defeito é hoje.

🪦

Ciclo de vida

Marca animais como falecidos (com data) ou arquivados. Ficam escondidos por defeito mas sempre acessíveis.

🌓

Tema claro & escuro

Segue o tema do sistema. Material 3 com paleta adaptada para ambos os modos.

Como instalar

Precisas do Flutter 3.19+ e do Dart 3.3+. Android compila em Windows / macOS / Linux; iOS só compila em macOS com Xcode.

1

Clonar

git clone https://github.com/celiaagoncalves/pet-health-app.git
cd pet-health-app
2

Preparar

flutter create . `
  --org com.celiagoncalves `
  --platforms ios,android
flutter pub get

As pastas ios/ e android/ não estão em git — geram-se aqui.

3

Correr

flutter run -d android
# ou
flutter run -d ios

Abre num emulador ou num telemóvel ligado. Sem device, usa flutter run e escolhe.

Privacidade. A base de dados Hive vive na pasta de documentos da app no próprio dispositivo. Nenhum dado é enviado para servidores externos. Não há contas, não há telemetria, não há tracking. Reinstalar a app apaga todos os dados — backup futuro está na roadmap.
